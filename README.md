* How to setup

- Installation without venv
pip install -r requirements.txt

- MACOS
python -m venv venv
source venv/bin/activate 
pip install -r requirements.txt

- WINDOWS 
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt

- Usage
python IDS.py {path_to_Events.txt} {path_to_Stats.txt} {days}
