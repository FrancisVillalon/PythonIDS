## How to Setup

### Installation without venv
```sh
pip install -r requirements.txt
```

### MACOS
```sh
python -m venv venv
source venv/bin/activate 
pip install -r requirements.txt
```

### WINDOWS
```sh
python -m venv venv
.\venv\Scripts\activate
pip install -r requirements.txt
```

## Usage
```sh
python IDS.py {path_to_Events.txt} {path_to_Stats.txt} {days}
```
